## 💻 Tech Stack & Features

*   **Framework:** React 18 with Vite & TypeScript
*   **Styling:** Tailwind CSS (Modern, high-contrast, fully responsive layouts)
*   **Animations:** Smooth state transitions, interactive drawers, and modal popups powered by `motion`
*   **Interactive Components:** 
    *   **Interactive Before/After Slider:** Allows users to slide and swipe dynamically to compare project transformations.
    *   **Filtered Map System:** Integrated South Carolina service area map highlighting Columbia, Lexington, West Columbia, and Richland County with custom location markers.
    *   **Fully-Formed Client Enquiries:** Secure front-end validation for quote requests and inspection scheduling.
*   **Iconography:** Beautiful, lightweight vector layouts powered by `lucide-react`

## 🚀 Getting Started (Development)

### Prerequisites

*   **Node.js** (v18 or higher recommended)
*   **npm** (comes packaged with Node.js)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/NoLabelSecurity/Wright-Way-Services-LLC.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Wright-Way-Services-LLC
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Local Development

To run the development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to preview the site.

### Build & Production Deploy

To compile a highly optimized, production-ready bundle of the static application inside the `dist` directory:
```bash
npm run build
```
